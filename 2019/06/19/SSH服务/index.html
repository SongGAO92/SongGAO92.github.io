SSH服务

2019-06-19

1更新源

sudo apt-get update

2 安装ssh服务

sudo apt-get install openssh-server

3 检测是否已经启动

ps -e | grep ssh
如果有ssh字样，说明已启动，如果没有就手动启动
/etc/init.d/ssh start

4 配置ssh-server,配置文件位于/etc/ssh/sshd_config,默认端口22，

为了安全，一般自定义为其他端口，然后重启
sudo /etc/init.s/ssh restart
